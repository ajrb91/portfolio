### **1. Practical Demos**

Introduction to bot structure, basic commands, log messages, and simple UI interactions.

- Reading spreadsheets

#### **demo-05-pdf-email-automation**
An end-to-end flow combining:
- PDF extraction
- String manipulation  
- Email commands  
- Automated notifications
