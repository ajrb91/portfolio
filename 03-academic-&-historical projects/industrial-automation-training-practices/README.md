# ⚙️ Industrial Automation Laboratory Practices

**Description:**  
Laboratory guide developed as part of an undergraduate thesis project at **Universidad del Zulia**, focused on **industrial automation networks** and the integration of **PLCs Siemens S7-1200** and **Allen Bradley ControlLogix** using **LabVIEW** as an HMI development platform.

**Authors:** Amílcar José Rodríguez Barrios, Eduardo José Socorro Acosta  
**Year:** 2016  
**Technology:** PLC Programming (Siemens S7-1200, Allen Bradley ControlLogix), LabVIEW, OPC Server, PROFINET, Ethernet/IP  
**Application Area:** Industrial Automation and Control Systems  
**Purpose:** Educational guide aimed at teaching students practical skills in PLC communication, HMI design, and industrial network integration.
